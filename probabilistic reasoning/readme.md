# assignment 3 — probabilistic reasoning

written and computational analysis of Bayesian networks and hidden Markov models.

## problems covered

- **Bayesian networks** — computing joint and conditional probabilities over Boolean variables using the chain rule and network structure; normalization trick for conditional inference
- **variable elimination** — step-by-step elimination of hidden variables to answer queries efficiently
- **hidden markov models** — forward algorithm for computing the probability of an observation sequence given a model

## approach

problems are solved analytically with full probability derivations. variable elimination is traced through each factor operation explicitly.
